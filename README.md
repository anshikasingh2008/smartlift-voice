# smartlift-voice
SmartLift Voice is an intelligent elevator control system that uses speech recognition, natural language processing, and machine learning to interpret voice commands and control elevator operations without physical contact. Designed for accessibility - helping elderly, visually impaired, and differently-abled individuals navigate buildings independently.  Features: • Voice command recognition ("Floor 5", "Open door") • Text-to-speech feedback • Web dashboard for monitoring • Emergency mode activation • Command history logging • Cross-platform (Windows, Linux, Raspberry Pi)  Tech Stack: Python, Flask, SpeechRecognition, pyttsx3, scikit-learn, NumPy
